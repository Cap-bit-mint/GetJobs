# JD 示例

本文档提供各类岗位的 JD 示例，用于测试和参考。

## 1. 产品经理 JD

```
【职位】高级产品经理
【公司】某互联网公司
【地点】北京
【薪资】35-60K·15薪

岗位职责：
1. 负责企业客户订单管理产品的需求分析与迭代
2. 主导产品规划，独立输出 PRD 与原型
3. 协调研发、设计、测试团队，推动产品按时上线
4. 基于数据分析，持续优化产品体验

任职要求：
1. 本科及以上学历
2. 5 年以上 B 端产品经验，有订单/ERP 系统经验优先
3. 熟练使用 SQL 进行数据分析
4. 优秀的跨团队沟通和推动能力
5. 有电商、SaaS 或企业服务背景优先

加分项：
- 有从 0 到 1 产品经验
- 有数据驱动决策经验
- 有带团队经验
```
