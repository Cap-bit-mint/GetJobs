# 中国大陆招聘语境

本文档说明 GetJobs 在中国大陆招聘场景下的特定处理规则。

## 招聘链路

### 典型流程

```
JD 发布 → ATS 筛选 → HR 初筛 → 业务面试 → 终面/HR谈薪 → Offer
```

### 各环节关注点

| 环节 | 关注点 | GetJobs 处理 |
|------|--------|--------------|
| ATS 筛选 | 关键词匹配、硬性条件 | ATS 评分、关键词自然嵌入 |
| HR 初筛 | 方向、年限、公司、薪资 | HR 吸引力评分、风险提示 |
| 业务面试 | 真实能力、项目细节、决策过程 | 面试准备评分、追问生成 |
| 终面 | 文化匹配、职业规划、稳定性 | 风险提示、沟通话术 |

---

## 国内平台特点

### 主流平台

| 平台 | 特点 | 使用建议 |
|------|------|----------|
| Boss 直聘 | 沟通直接、反馈快 | 开场白简洁（50字内） |
| 拉勾 | 互联网岗位多、评价丰富 | 可参考公司评价 |
| 猎聘 | 中高端职位多 | 简历可适当突出 |
| 智联招聘 | 综合类、量大 | 标准简历即可 |
| 前程无忧 | 传统行业多 | 注意格式兼容性 |
| 脉脉 | 社交属性强 | 可辅助了解公司 |

### 平台差异

| 场景 | 建议 |
|------|------|
| Boss 直聘 | 开场白短、有具体匹配点、主动提问 |
| 猎头推荐 | 简历突出核心优势、配合猎头话术 |
| 官网投递 | ATS 友好版、标准格式 |
| 校招 | 突出学习能力、项目、实习 |

## 简历格式规范

### 国内通用规范

| 要素 | 建议 |
|------|------|
| 页数 | 应届/1-3年：1页；3-5年：1-2页；5年+：2页 |
| 照片 | 可选，非必须 |
| 年龄 | 可选，非必须 |
| 婚育 | 可选，非必须 |
| 期望薪资 | 可写范围或"面议" |
| 时间线 | 倒序排列 |

### ATS 友好格式

- 使用标准标题（教育背景、工作经历、项目经历、技能）
- 避免表格、图片、文本框
- 日期格式统一（2022.01 或 2022.01-2024.06）
- 避免特殊符号和花哨排版

## JD 解读

### 常见 JD 话术

| JD 表述 | 真实含义 | 建议 |
|----------|----------|------|
| "熟悉/了解" | 最好有实际使用经验 | 可写，但需有项目证据 |
| "优先/加分" | 非必须，但有优势 | 有则强调，无则不写 |
| "有XXX经验优先" | 硬性条件可能较弱 | 可尝试投递 |
| "面议" | 薪资可谈 | 可写期望范围 |
| "弹性工作" | 可能有加班 | 心理准备 |
| "扁平管理" | 团队小或初创 | 了解公司阶段 |

### 硬性条件识别

| 条件类型 | 标识词 | 建议 |
|----------|--------|------|
| 必须满足 | 必须、必备、至少 | 不满足不建议投递 |
| 强烈建议 | 优先、有经验者优先 | 不满足需突出其他优势 |
| 加分项 | 加分、优先、熟悉者优先 | 可不满足 |

## 面试特点

### 常见面试轮次

| 轮次 | 内容 | 关注点 |
|------|------|----------|
| HR 面 | 基本情况、求职动机、薪资 | 稳定性、匹配度 |
| 技术/业务面 | 项目细节、专业能力 | 真实性、深度 |
| 主管面 | 决策能力、团队协作 | 潜力、文化匹配 |
| HR 终面 | 薪资谈判、福利 | 期望管理 |

### 常见问题

| 问题类型 | 示例 | 回答要点 |
|----------|------|----------|
| 离职原因 | 为什么要离职？ | 正面表达、不抱怨前公司 |
| 职业规划 | 未来3-5年规划？ | 与申请岗位一致 |
| 薪资期望 | 期望薪资？ | 提前调研、给范围 |
| 项目细节 | 具体做了什么？ |STAR 法则、量化成果 |
| 团队管理 | 带多少人？ | 如实说明、强调协作 |

## 转行/转岗

### 国内特点

- 国内对转行容忍度相对较低
- 需要突出可迁移能力
- 最好有学习/项目证明
- 解释清楚转型动机

### 处理建议

| 场景 | 建议 |
|------|------|
| 完全转行 | 强调可迁移能力、提供学习证据 |
| 转岗（同类） | 突出相关经验、降低学习成本 |
| 跨行业转岗 | 强调方法论迁移、行业适应 |

---

## 特殊注意事项

### 简历中应避免

- 政治观点
- 宗教信仰
- 健康状况（非相关岗位）
- 婚姻状况（非相关岗位）
- 前公司负面信息

### 信息脱敏建议

| 信息类型 | 建议处理 |
|----------|----------|
| 手机号 | 可写，便于联系 |
| 邮箱 | 建议写 |
| 身份证号 | 不写 |
| 家庭住址 | 写城市即可 |
| 紧急联系人 | 不写 |
| 薪资 | 可写历史范围或期望 |

---

## 地域差异

| 地区 | 特点 |
|------|------|
| 北上广深 | 竞争激烈、薪资较高 |
| 杭州/成都/武汉 | 互联网发展快、生活成本适中 |
| 二线城市 | 机会增加、稳定为主 |

---

## 风险提示

- 简历信息与实际情况必须一致
- 背调可能核实工作经历
- 薪资流水可能要求提供
- 学信网可能验证学历
- 部分岗位需要离职证明
